## 変換後の形式（H形鋼の例）

const SECTION_DATA = {
  'H形鋼': {
    type: 'H',
    label: 'H形鋼',
    jis: 'JIS G 3192',
    jisSub: 'Hot-rolled H beams',
    specs: [
      { name:'H-100×50',  H:100, B:50,  t1:5, t2:7, r:8, Ac:11.85, W:9.30,  Ix:187,  Iy:14.8, Zx:37.5, Zy:5.91, ix:3.98, iy:1.12 },
      { name:'H-100×100', H:100, B:100, t1:6, t2:8, r:8, Ac:21.59, W:16.9,  Ix:378,  Iy:134,  Zx:75.6, Zy:26.7, ix:4.18, iy:2.49 },
    ]
  }
};

## 各キーの意味
- name : 規格名称（例: "H-100×50"）
- H    : 全高 (mm)
- B    : フランジ幅 / 辺幅 (mm)
- t1   : ウェブ厚 (mm)
- t2   : フランジ厚 (mm)
- r    : フィレット半径 (mm)
- Ac   : 断面積 (cm²)
- W    : 単位質量 (kg/m)
- Ix   : 断面二次モーメント X軸 (cm⁴)
- Iy   : 断面二次モーメント Y軸 (cm⁴)
- Zx   : 断面係数 X軸 (cm³)
- Zy   : 断面係数 Y軸 (cm³)
- ix   : 断面二次半径 X軸 (cm)
- iy   : 断面二次半径 Y軸 (cm)

## 依頼内容
下記に貼り付けるJIS表のデータを、上記と同じJavaScriptオブジェクト形式に変換してください。
- キー名は上記の通りに統一すること
- 数値は単位を除いた数字のみ（例: 9.30 ← "9.30 kg/m"ではなく）
- name は "鋼材記号-サイズ" の形式（例: "H-100×50"）
- 出力はJavaScriptコードのみ、説明文は不要
