# CODEX指示：新規格追加（角パイプ・C形鋼・軽量溝形鋼）

## 概要

`calc.js` の `STEEL` オブジェクトに以下9カテゴリ・計222件を追加する。
既存カテゴリは**一切変更しない**。

## 命名規則

| カテゴリ | プレフィックス | 例 |
|---|---|---|
| 角パイプ（正方形・長方形） | `p-` | `p-50×50×1.6` |
| R付き角パイプ（丸コーナー） | `Rp-` | `Rp-40×40×1.6` |
| C形鋼（軽量） | `C-` | `C-100×50×20×2.3` |
| 軽量溝形鋼 | `[-` | `[-100×40×3.2` |

寸法順：角パイプ＝`幅×高さ×厚さ`、C形鋼＝`高さ×幅×リップ×厚さ`、軽量溝形鋼＝`高さ×幅×厚さ`

---

## 対象ファイル

`calc.js`

---

## 挿入位置

`var STEEL = {` ブロック内の `'丸鋼':[...]` の直後に追加する。

```
  '丸鋼':[
    ...
    ['RB-250',385.336]
  ],           ← このカンマの後に下記を全部追加
  '中径角パイプ（正方形）':[
    ...
  ],
  ...
  '軽量溝形鋼':[
    ...
  ]            ← 最後はカンマなし
};
```

## 動作確認

ブラウザをリロードし重量タブの検索欄で以下を確認：

- `p` と入力 → 角パイプ各種が候補に出る
- `p-100×100` と入力 → 100角パイプが絞り込まれる
- `[` と入力 → 軽量溝形鋼が候補に出る
- `C-100` と入力 → C形鋼（軽量）が出る
- 選択して＋追加 → 正しいkg/mで重量計算される
