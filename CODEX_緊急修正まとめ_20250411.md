# 緊急バグ修正まとめ（2025-04-11）

このCODEXはClaudeが直接適用済みです。参照・確認用として残します。

---

## 問題の概要

複数のJSファイルとHTMLが途中で切れていた（ファイル破損）ため、
ブラウザコンソールに `SyntaxError: Unexpected end of input` が連発し、
重量タブの鋼材検索・選択が一切できない状態になっていた。

---

## 修正① `index.html` — scriptタグが途切れていた

**症状：** 820行目で `<script src="` の途中で切断。
`final-overrides.js`・`weight.js`・`data.js` が一切ロードされていなかった。

**修正：** 以下を末尾に追加して `</body></html>` で閉じた。

```html
  <script src="final-overrides.js"></script>
  <script src="weight.js"></script>
  <script src="data.js"></script>
</body></html>
```

---

## 修正② `index.html` — CO₂排出量・重量逆算のwip-cardを削除

重量タブ左サイドバーに残っていた以下のブロックを削除した（以前に削除済みのはずだった）。

```html
<!-- CO₂排出量（準備中） -->
<div class="wip-card" style="margin-top:10px"> ... </div>

<!-- 重量逆算（準備中） -->
<div class="wip-card"> ... </div>
```

---

## 修正③ `weight.js` — wCmdFilter関数が途切れていた

**症状：** 657行目で `it.spec.replace(/[^0-9]/g,'').indexOf(q.re` の途中で切断。
`wCmdSelect`・`wCmdOutside`・`wCmdKey` も丸ごと欠落。
→ 検索欄に入力しても候補を選択できない。

**修正：** 切断箇所から `wCmdFilter` を完成させ、以下4関数を追加した。

- `wCmdFilter` — 検索・グループ表示・ドロップダウン制御
- `wCmdSelect(it)` — 候補選択時にkind/spec/kgm欄を更新
- `wCmdOutside(e)` — 外クリックでドロップダウンを閉じる
- `wCmdKey(e)` — ↑↓/Enter/Escキー操作

---

## 修正④ `data.js` — drawHBeamSVG関数が途切れていた（最大の原因）

**症状：** 112行目で `return \`<svg width="${viewW}" height="${viewH}" view` の途中で切断。
→ `data.js` をロードするだけでSyntaxError → 重量タブの `wInit` も連鎖エラー。

**修正：** git HEADのバージョンを元に完全復元し、同時に山形鋼CODEXも適用した。

復元内容：

- `dataInit`・`renderDataKindTabs`・`renderDataSpecPicker`
- `filterDataSpecOptions`・`renderDataSpec`・`selectDataSpec` など全関数
- `drawLAngleSVG` 関数（山形鋼CODEX変更①）
- `SECTION_DATA['山形鋼']` データ14サイズ（山形鋼CODEX変更②）
